# IT Support Lab Environment

## Overview

This project simulates a real-world IT support environment where common technical issues related to systems, software, networking, and user tools are practiced and resolved.

The purpose of this lab is to develop hands-on IT infrastructure and troubleshooting skills relevant to entry-level IT support roles.

---

## Technologies Used

* Windows Operating System
* Ubuntu Linux (Virtual Machine)
* VirtualBox
* Microsoft 365 (Outlook, Teams, OneDrive)
* Command Line Tools
* Git & GitHub

---

## Key Tasks Performed

### System Setup

* Installed Ubuntu Linux using VirtualBox
* Configured system settings and user accounts
* Managed software installation and updates

### Troubleshooting Practice

* Resolved slow system performance issues
* Fixed software installation errors
* Handled file permission problems

### Networking Basics

* Checked IP configuration using ipconfig / ifconfig
* Tested internet connectivity using ping
* Simulated network troubleshooting scenarios

### Microsoft 365 Support Practice

* Used Outlook and Teams for communication scenarios
* Practiced file sharing and syncing via OneDrive
* Simulated login and synchronization issues

---

## Learning Outcomes

* Improved practical troubleshooting skills
* Better understanding of system environments
* Basic networking knowledge
* Experience documenting IT solutions
