# it-support-lab-environment

### 🖥️ IT Support Lab Environment

This project simulates a real-world IT support environment to practice troubleshooting, system setup, and basic IT infrastructure tasks.

The goal is to develop hands-on skills required for IT support roles, including system configuration, issue resolution, networking basics, and user support scenarios.

---

### 🎯 Objectives

* Practice real-world IT support tasks
* Troubleshoot hardware, software, and connectivity issues
* Work with Windows and Linux environments
* Understand basic networking and cloud tools
* Document IT processes and solutions

---

### 🧰 Tools & Technologies

* Windows OS
* Linux (Ubuntu via VirtualBox)
* VirtualBox
* Microsoft 365 (Outlook, Teams, OneDrive)
* Command Line (Linux/Windows)
* Git & GitHub

---

### 🔧 Tasks Performed

#### 🖥️ System Setup

* Installed and configured Ubuntu using VirtualBox
* Set up development and system environments
* Managed user accounts and system settings

---

#### 🛠️ IT Troubleshooting

* Resolved software installation errors
* Diagnosed system performance issues
* Fixed permission and access-related problems

---

#### 🌐 Networking Basics

* Checked IP configurations using `ipconfig` / `ifconfig`
* Tested connectivity using `ping`
* Troubleshot network-related issues

---

#### ☁️ Microsoft 365 Support

* Practiced Outlook, Teams, and OneDrive usage
* Simulated user issues like login and sync errors
* Managed file sharing and access

---

#### 📋 Documentation

* Created step-by-step troubleshooting guides
* Documented common IT issues and solutions
